# Gurmaanpreet Singh - Portfolio

Welcome to the repository for my personal portfolio! This website is built to showcase my top projects, skills, and experience as a Computer Science & Data Science student. 

## 🔗 Live Site
[https://gurmaan.vercel.app](https://gurmaan.vercel.app)

## 🛠️ Built With
This portfolio is built with modern web technologies:
* **React 18** - UI Library
* **Vite** - Build Tool
* **Framer Motion** - Animations
* **Vanilla CSS** - Styling

## 🎯 Features
* **Minimalist & Professional Design:** Focus on typography, clean layout, and readability.
* **Projects Showcase:** Highlight my data science and software engineering projects.
* **Auto-Deployed:** Automatically deployed using Vercel from this GitHub repository.

## 💻 Local Setup
If you would like to run this site locally:

1. Clone the repository: 
```bash
git clone https://github.com/GurmaanSidhu/portfolio.git
```
2. Install dependencies:
```bash
npm install
```
3. Start the development server:
```bash
npm run dev
```
